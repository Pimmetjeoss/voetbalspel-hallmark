# Voetbalspel Hallmark prototype

Mobielvriendelijke statische prototype-site voor een voetbalvoorspelspel.

## Hallmark

Gebouwd met Hallmark-principes:

- Macrostructure: Workbench
- Genre: playful / sport
- Theme: studied-DNA, Refero Caldera
- Nav: N7 brutal slab
- Footer: Ft8 marquee scroll

Auditrapport: `.hallmark/audit-2026-05-20.md`.

## Lokaal draaien

```bash
cd site
python -m http.server 8123
```

Open daarna `http://127.0.0.1:8123/`.
