Opdracht: Website voor Voetbalspel

Ik zoek een eenvoudige maar aantrekkelijke website voor een voetbalvoorspelspel.

Functioneel moet de website kunnen:
- Gebruikersregistratie / inloggen
- Uitslagen van voetbalwedstrijden invoeren door beheerder
- Automatische puntentelling
- Ranglijsten: week, maand, algemeen
- Eenvoudig beheerpaneel voor beheerder
- Aparte pagina voor artikelen/updates
- Visuele prijzenkast
- Mobielvriendelijk

Designrichting:
Gebruik Caldera/Refero DNA: pixelated cyber-playground, muted basalt canvas, ash white rounded cards, digital orange CTA's, ultra-bold condensed headings, pixel/dot texture. Vertaal dit naar voetbal/game-context; niet letterlijk Caldera kopiëren.

Hallmark discipline:
- Pre-flight: als projectbestanden bestaan, eerst lezen.
- Macrostructure: Workbench + Scoreboard Grid.
- Geen fake metrics/testimonials.
- Mobile-first: prediction cards en leaderboard moeten uitstekend werken op 320/375/414/768 px.
- Gebruik tokens; geen losse hex/font-values buiten tokens.
- Bouw eerst homepage + dashboard + admin + updates + prijzenkast als routes of duidelijke componenten.
- Admin mag simpel zijn; focus op lage complexiteit en duidelijke UX.
